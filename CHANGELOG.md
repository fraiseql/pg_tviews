## [0.1.0-beta.1] - 2025-12-13

### Added
- Initial public API for TVIEW conversion
- Incremental refresh with transaction queue
- Dependency graph analysis
- PostgreSQL 13-18 support
- Basic TVIEW metadata introspection
- Comprehensive testing framework (security, performance, failure modes)
- API stability classifications and registry
- Memory profiling and leak detection tools
- Performance regression testing framework

### Known Limitations
- Queue debugging functions experimental
- Performance claims validated but framework-ready
- Limited error path testing in beta
- No deprecation policy (beta period)
